# Event Booking System

## Project Overview
This project is a simple event booking system where users can view events, book tickets, and manage their bookings.

## Features
- Event listing with details
- Booking functionality
- Booking confirmation page
- Responsive design using Bootstrap
- Client-side form validation
- Local storage for session data
- Advanced features including calendar view, search functionality, and modals

## Libraries and Tools
- **Moment.js**: For date manipulation.
- **Bootstrap**: For responsive design.
- **JSON Server**: To simulate backend for event data.

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the JSON server using `json-server --watch events.json`.
4. Open `index.html` in your browser.

## Folder Structure
- `index.html`: Home page
- `event-details.html`: Event details page
- `booking-confirmation.html`: Booking confirmation page
- `styles.css`: CSS styling
- `script.js`: JavaScript functionality
- `events.json`: Mock event data
